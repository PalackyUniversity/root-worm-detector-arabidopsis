## Installation
1. Clone the repository
2. Install python3
```bash
sudo apt-get install python3 python3-pip
```
3. Install the required packages
```bash
pip3 install -r requirements.txt
```

## Run
1. Images should be placed in `data` directory
2. Run Python script:
```bash
python3 predict.py
```
3. Output is in `results` directory
